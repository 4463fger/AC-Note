## 🧶 整体结构（像整理毛线球一样整齐喵）

```csharp
public class Code01_Bitset
{
    private int[] set; // 用int数组当储物柜的抽屉哟
    //...
}
```
每个int就像有32个小抽屉的储物盒（因为int有32位喵！）

---

## 🎀 构造函数（准备储物柜）

```csharp
public Code01_Bitset(int n)
{
    set = new int[(n + 31) / 32]; 
}
```
(a + b -1)/ b => 实现向上取整

例子喵：
- 当n=5时 → (5+31)/32=1 → 需要1个int
- 当n=40时 → (40+31)/32=2 → 需要2个int

---
## 🎈 Add方法（贴上小星星⭐）

```csharp
set[num / 32] |= 1 << (num % 32);
```

就像在储物盒的对应位置贴星星：
- num=5 → 第0个int的第5位
- `1<<5` = 00100000
- 用`|`贴上小星星~

---

## 🧹 Remove方法（打扫灰尘啦）

```csharp
set[num / 32] &= ~(1 << (num % 32));
```
就像用吸尘器吸走指定位置的灰尘：
- 先做`~`反转换上吸尘头
- 用`&`打扫干净~

---

## 🎭 Reverse方法（变魔术啦！）

```csharp
set[num / 32] ^= 1 << (num % 32);
```
像翻转小贴纸：
- 原来有星星 → 变空白
- 原来空白 → 贴星星
- 用`^`魔法就能做到啦~

---

## 🔍 Contains方法（找找小星星）

```csharp
return ((set[num / 32] >> (num % 32)) & 1) == 1;
```
步骤喵：
1. 把目标位移到最低位（右移）
2. 用`&1`看看是不是亮晶晶的小星星~

---

## 🍭 总结表格（甜甜的糖果版）

| 方法       | 操作符 | 效果               | 类比           |
|------------|--------|--------------------|----------------|
| Add        | `|`    | 贴星星⭐          | 用荧光笔标记   |
| Remove     | `&~`   | 擦除指定位置       | 用橡皮擦干净   |
| Reverse    | `^`    | 翻转当前状态       | 开关小夜灯     |
| Contains   | `&1`   | 检查是否有星星     | 用放大镜查看   |
